About WebHACC...

This checker is OBSOLETE. See manakai project Web site for new versions of checkers.

WebHACC is a conformance checker (or validator) for Web documents, supporting latest Web standards including HTML5, Atom 1.0, CSS3.

Powered by ...

  • manakai, a Perl library for DOM.
  • Whatpm, a subproject of manakai that supports HTML, XML, CSS, and other Web languages.
  • Regexp::Parser and its extensions for JavaScript regular expressions.

Source code

WebHACC is written in pure Perl, using aforementioned modules.

You can clone the Git repository (GitHub / Suika) of the project by the following command: 

$ git clone https://github.com/wakaba/webhacc.git
$ git submodule update --init
