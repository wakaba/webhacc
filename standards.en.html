CSS

The current implementation assumes that there are following requirements:

  • The charset parameter of the media type text/css MUST be treated as if it were an unknown parameter for the purpose of determining styling langauge for an HTML style element.
  • Content of an HTML style element whose content is interpreted as a CSS style sheet MUST consist of zero or more text nodes. Note that it may also contain zero or more comment and/or processing instruction nodes.
